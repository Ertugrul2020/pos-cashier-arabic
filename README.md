
PosAppArabic V2 - POS بالعربية (يدعم الطباعة الحرارية 58mm و80mm)

المحتوى:
- إدارة منتجات (Room)
- مسح باركود (ZXing)
- بيع نقدي
- طباعة إيصال حراري بالعربية (تحويل نص إلى صورة + ESC/POS library)

لبناء APK محليًا: افتح المشروع في Android Studio ثم Build > Build APK(s).
لبناء APK سحابياً استخدم GitHub Actions كما في المشروع السابق.

ملاحظات الطباعة:
- الكود يدعم الطابعات الحرارية الشائعة عبر مكتبة DantSu ESC/POS.
- يتم تحويل النص العربي إلى Bitmap (دعم RTL) ثم إرسال للطابعة.
